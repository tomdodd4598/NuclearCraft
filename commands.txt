 NuclearCraft Server Commands
#============================#

Required arguments are labelled <...>. Optional arguments are labelled [...].

---------------------------------------------------------------------------------------------------------------------------

 Set Chunk Radiation: /nc_set_chunk_radiation <radiation_level> [coord_x] [coord_z]
>--------------------

 Set Chunk Radius Radiation: /nc_set_chunk_radius_radiation <radiation_level> <chunk_radius>
>---------------------------

 Set World Radiation: /nc_set_world_radiation <radiation_level>
>--------------------

 Set Player Radiation: /nc_set_player_radiation <radiation_level> [player]
>---------------------
